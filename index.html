Cómo publicar en Vercel
  1. Descomprime el ZIP que descargaste y entra a la carpeta
  2. Sube el código a un repositorio en github.com
  3. Entra a vercel.com → New Project → conecta tu repo
  4. Framework Preset: Other — no cambies nada más
  5. Clic en Deploy — en ~30 seg tienes tu URL
  6. Copia la URL y ponla en tu bio de X
Variables de entorno (opcional)
Si quieres que la app se pre-cargue con tus credenciales para otros usuarios, puedes configurarlas en Vercel → Settings → Environment Variables:

X_API_KEY=tu_api_key
X_API_SECRET=tu_api_secret
X_ACCESS_TOKEN=tu_access_token
X_ACCESS_SECRET=tu_access_secret
X_USERNAME=tu_usuario
